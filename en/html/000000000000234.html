Hood Release Handle Replacement
  1. Remove the kick panel.
  1. Remove the bolt, and release the hook (A), then remove the hood release handle (B) from the body.
  1. Disconnect the hood opener cable (C) from the hood release handle. Take care not to kink the cable.
  1. Install the hood release handle in the reverse order of removal, and note these items:
    • Make sure the hood opener cable is connected properly.
    • Make sure the hood opens properly.
