A/T 4th and 5th Clutch Reassembly
Special Tools Required

Clutch spring compressor set
 07LAE-PX40000

Clutch spring compressor attachment
 07LAE-PX40100

Clutch spring compressor attachment
 07HAE-PL50101

Clutch spring compressor bolt assembly
 07GAE-PG40200

Snap ring pliers
 07LGC-0010100
  1. Soak the clutch discs thoroughly in ATF for a minimum of 30 minutes.
  1. Install the new O-rings (A) on the clutch piston (B).
  1. Install the clutch piston (A) in the clutch drum (B) while applying pressure and rotating to ensure proper seating. Do not pinch the O-ring.
  1. Install the return spring (A) and spring retainer (B), and position the snap ring (C) on the spring retainer.
  1. Install the special tools.
  1. Be sure the special tool (A) is adjusted to have full contact with the spring retainer (B).
  1. If either end of the special tool is set over an area of the spring retainer that is unsupported by the return spring, the retainer may be damaged.
  1. Compress the return spring until the snap ring can be installed.
  1. Install the snap ring with the snap ring pliers.
  1. Remove the special tools.
  1. Install the waved spring (A) in the 4th clutch drum (B). Starting with the clutch wave-plate, alternately install the wave-plates (C) (4) and discs (D) (4). Install the clutch-end-plate (E) with the flat side down on the top disc.
  1. Install the waved spring (A) in the 5th clutch drum (B). Starting with the clutch wave-plate, alternately install the wave-plates (C) (4) and discs (D) (4). Install the clutch-end-plate (E) with the flat side down on the top disc.
  1. Install the snap ring with a screwdriver to secure the clutch-end-plate.
  1. Check that the clutch piston moves by applying air pressure into fluid passage.
