M/T Assembly Installation
Special Tools Required

Engine support hanger
 AAR-T1256-J00

This special tool is available from Snap-on Tools International, LLC.
NOTE: Use fender covers to avoid damaging painted surfaces.
  1. Apply the release fork bolt joint-hollow (A) of the release fork (B) with 1.0−1.6 grams (0.04−0.06 oz.) of Super High Temp Urea Grease (P/N 41211-PY5-305). Apply the slave cylinder push-rod joint-hollow (C) with 0.4−1.0 grams (0.01−0.04 oz.) of Urea Grease.
  1. Coat the inside of the release bearing (D) with 0.4−1.0 grams (0.01−0.04 oz.) of Urea Grease.
  1. Install the release bearing on the bearing guide (E) while inserting the release fork through the hole (F) of the clutch housing. Align the joint-hollow of the release fork with the release fork bolt top (G), and secure the release fork with the set spring (H).
  1. Install the release fork boot (I) as to seal around the release fork and clutch housing.
  1. Install the dowel pins in the clutch housing.
  1. Place the transmission on the jack, and raise the transmission to the engine level.
  1. Install the lower transmission housing mounting bolt (A) part-way in the bolt hole on the engine, then attach the transmission to the engine.
  1. Install the lower transmission housing mounting bolt (B), and tighten the bolts.
  1. Install the rear transmission housing mounting bolt.
  1. Install the front transmission housing mounting bolt.
  1. Install the upper transmission housing mounting bolts.
  1. Secure the transmission mount bracket on the transmission housing with the new mounting bolts.
  1. Install the clutch housing cover.
  1. Install the intermediate shaft and lower torque rod bracket.
  1. Install the driveshafts in the differential and over the intermediate shaft end.
  1. Support the front subframe with a wood piece and a jack, and lift it up to the body.
  1. Loosely install the new subframe mounting bolts.
  1. Align the reference marks (A) with the center of the subframe mounting bolts (B), then tighten the mounting bolts on the subframe to the specified torque.
  1. Secure the lower torque rod with the new bolts.
  1. Connect the ball joints (A) to both lower arms (B), and install the new ball joint nuts (C). Tighten the nuts, then secure the nuts with the spring clips (D). Install the spring clips from the inside of the vehicle, with their hooked sides facing the front (E) of the vehicle in the direction shown.
  1. Connect the stabilizer links (F) to the lower arms, and install the nuts (G). Insert a 5mm Allen wrench (H) in the top of the ball joint pins (I), and tighten the nuts.
  1. Connect the headlight adjuster leveling sensor connector (if equipped).
  1. Remove the hanger plate (A) and bolt (B), and install the harness cover bracket (C) on the engine.
  1. Remove the engine support hanger.
  1. Install the air cleaner housing bracket (A).
  1. Install the harness cover (B) on its bracket, and secure it with the bolts.
  1. Secure the connector bracket (C).
  1. Connect disconnected connectors (D) and vacuum tube (E), and secure the tube with the clip.
  1. Install the ACG terminal (A), connect the connectors (B), and install the harness clamp (C) on its bracket (D).
  1. Secure the harness cover with the bolts.
  1. Install the engine cover.
  1. Apply Super High Temp Urea Grease (P/N 41211-PY5-305) to the top of the slave cylinder push-rod (A), and install the slave cylinder (B) and clutch line (C).
  1. Connect the gearshift cable ends (A) on the change lever joints (B), and secure the cable bracket (C) with the bolts.
  1. Insert the lock pins (D) through the hole of the change lever joint rods so that the hooked end of the lock pin snaps into the countersunk of the joint rod.
  1. Connect the back-up light switch connector (A), output shaft (countershaft) speed sensor connector (B), and reverse lockout solenoid connector (C).
  1. Install the air cleaner housing bracket.
  1. Refill the transmission with MTF.
  1. Install the battery base and resonator.
  1. Install the air cleaner housing and intake air duct.
  1. Install the front bulkhead cover.
  1. Install the cowl cover.
  1. Install the windshield wiper arms.
  1. Install the battery tray, battery, and battery cover, then secure the battery with its hold-down bracket.
  1. Connect the battery terminals, and apply grease around the battery terminals.
  1. Check the clutch operation.
  1. Check and adjust the front wheel alignment.
  1. Install the splash shield.
  1. Test-drive the vehicle.
  1. After the test drive the vehicle, raise the vehicle on a lift, and make sure it is securely supported.
  1. Loosen the upper torque rod bolt (A).
  1. Loosen the transmission mount base bracket bolt (A) and nuts (B).
  1. Loosen the lower torque rod bolts (A) (B).
  1. Retighten the lower torque rod bolts (B) to the specified torque, then retighten the bolt (A) to the specified torque.
  1. Tighten the transmission mount base bracket bolt and nuts to the specified torque.
  1. Tighten the upper torque rod bolt to the specified torque.
