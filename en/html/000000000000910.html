Radiator Fan Switch Replacement
  1. Drain the engine coolant.
  1. Remove the harness clamp, then remove the resonator.
  1. Disconnect the radiator fan switch connector, then remove the radiator fan switch (A).
  1. Install the radiator fan switch using a new O-ring (B).
  1. Install the resonator, then install the harness clamp.
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
