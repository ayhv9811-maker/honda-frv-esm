A/T Input Shaft (Mainshaft) Speed Sensor Replacement
  1. Remove the air cleaner housing and intake air duct.
  1. Disconnect the input shaft (mainshaft) speed sensor connector, and remove the input shaft (mainshaft) speed sensor (A).
  1. Install the new O-ring (B) on the new input shaft (mainshaft) speed sensor, then install the input shaft (mainshaft) speed sensor in the transmission housing.
  1. Check the connector for rust, dirt, or oil, then connect the connector securely.
  1. Install the intake air duct and air cleaner housing.
