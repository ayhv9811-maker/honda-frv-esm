A/T Shaft Assembly and Housing Installation
  1. Install the differential assembly in the torque converter housing.
  1. Install the baffle plate on the servo body.
  1. Assemble the mainshaft, countershaft, and secondary shaft.
  1. Join the mainshaft subassembly (A), countershaft subassembly (B), and secondary shaft subassembly (C) together, and install them in the torque converter housing. Do not bump the countershaft on the baffle plate (D).
  1. Make sure if the countershaft and differential are clear of the baffle plate.
  1. If the detent arm was removed, install the detent arm (A) with arm collar (B) on the servo body (C), and install the new lock washer (D) by aligning its cutout (E) with the projection (F) of the servo body. Install and tighten the bolt, then bend the lock tab of the lock washer against the bolt head.
  1. Install the control shaft (A) in the torque converter housing aligning the manual valve lever pin (B) on the control shaft with the guide of the manual valve (C). Pull the manual valve gently when aligning the manual valve with the control shaft.
  1. Hook the detent arm spring (A) to the detent arm (B).
  1. Turn the shift fork shaft (A) so the large chamfered hole (B) is facing the fork bolt hole (C) of the shift fork (D).
  1. Install the shift fork and reverse selector together on the shift fork shaft and countershaft. Secure the shift fork to the shift fork shaft with the lock bolt and a new lock washer (E), then bend the lock tab of the lock washer against the bolt head.
  1. Install the needle bearing and countershaft reverse gear on the countershaft.
  1. Install the reverse idler gear in the transmission housing.
  1. Install the idler gear shaft, if it was removed.
  1. Install the three dowel pins (A) and a new gasket (B) on the torque converter housing (C).
  1. Align the spring pin (D) of the control shaft (E) with the transmission housing groove (F) by turning the control shaft. Do not squeeze the end of the control shaft tips together when turning the shaft. If the tips are squeezed together, it will cause a faulty shift position signal or position due to the play between the control shaft and the transmission range switch.
  1. Place the transmission housing (G) on the torque converter housing. Do not install the input shaft (mainshaft) speed sensor and output shaft (countershaft) speed sensor before installing the transmission housing on the torque converter housing.
  1. While expanding the snap ring of the secondary shaft bearing using the snap ring pliers, push the transmission housing down to start the secondary shaft bearing through the snap ring. Then release the pliers, and push down the housing until it bottoms and snap ring snaps in place in the secondary shaft bearing snap ring groove.
  1. Verify that the secondary shaft bearing snap ring (A) is seated in the bearing and housing groove, and check that the ring end gap (B) is 0−7 mm (0−0.28 in.)
  1. Install the transmission housing mounting bolts (20 bolts) along with the transmission hanger. Tighten the mounting bolts to 44 N·m (4.5 kgf·m, 33 lbf·ft) in two or more steps in a crisscross pattern.
  1. Install the new O-ring (A) on the input shaft (mainshaft) speed sensor (B), and install the input shaft (mainshaft) speed sensor in the transmission housing.
  1. Install the new O-ring (C) on the output shaft (countershaft) speed sensor (D), and install the output shaft (countershaft) speed sensor and sensor washer (E).
  1. Install the new O-ring (F) on the shift solenoid harness connector (G), and install the shift solenoid harness connector in the transmission housing.
  1. Connect the harness terminals to the solenoids:
    • RED wire connector to shift solenoid valve E.
    • GRN wire connector to shift solenoid valve C.
    • ORN wire connector to shift solenoid valve B.
    • BLU wire connector to shift solenoid valve A.
    • YEL, WHT, WHT wire connector to shift solenoid valve D.
  1. Install the new gasket (A), dowel pins (B), and shift solenoid valve cover (C), and secure the cover with the bolts.
