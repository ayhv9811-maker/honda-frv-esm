Timing Belt Installation
NOTE:
  • If you are installing the auto-tensioner, refer to the auto-tensioner installation procedure.
  • If you are replacing the timing belt, refer to the timing belt replacement procedure .
  • Inspect the water pump before installing the timing belt.
  1. Clean the upper cover, lower covers, crankshaft, timing belt guide plate, timing belt drive pulley, and camshaft pulley.
  1. Install the timing belt guide plate (A) with the concave surface facing in. Install the new timing belt drive pulley (B).
  1. Set the crankshaft to top dead center (TDC). Align the TDC mark (A) on the timing belt drive pulley with the pointer (B) on the oil pump.
  1. Set the camshaft to TDC. The ‘‘UP'' mark (A) on the camshaft pulley should be at the top, and align the TDC marks (B) on the camshaft pulley with the top edge of the head.
  1. Install the timing belt in a counterclockwise sequence, starting with the drive pulley.
    -1.
    Drive pulley (A)
    -2.
    Tensioner pulley (B)
    -3.
    Water pump pulley (C)
    -4.
    Camshaft pulley (D)
  1. Install the crankshaft position (CKP) sensor.
  1. Install the bolts loosely, then tighten the side engine mount bracket mounting bolts in the numbered sequence shown.
  1. Install the lower cover.
  1. Install the crankshaft pulley
  1. Rotate the crankshaft pulley about five or six turns counterclockwise so that the timing belt positions on the pulleys.
  1. Turn the crankshaft pulley so its TDC mark (A) lines up with the pointers (B).
  1. Check the camshaft pulley marks (A).
    • If the camshaft pulley marks are also at TDC, go to Step 13 .
    • If the camshaft pulley marks are not at TDC, remove the timing belt and repeat Step 3 through 11.
  1. Install the upper cover (A), then connect the camshaft position (CMP) sensor connector (B), and install the grommet (C).
  1. Install the upper bracket, then tighten the nuts in the numbered sequence shown.
  1. Install the cylinder head cover
  1. Install the alternator
  1. Install the splash shield.
