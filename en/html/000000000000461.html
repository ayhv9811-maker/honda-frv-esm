Resetting the Power Window Control Unit (Master Switch)
Resetting the power window control unit is required after performing the following procedures:
  • Loss of battery power.
  • Loss of power from the No. 23 (20 A) fuses in the under-dash fuse/relay box.
  • Open circuit caused by disconnecting the 20P connector from the power window master switch.
  • Removal of the regulator, glass, or glass run channel.
  1. Make sure the glass is installed properly.
  1. Close the driver's door.
  1. Turn the ignition switch ON (II).
  1. Move the drivers window all the way down using the manual DOWN function of the driver's power window switch, and all the way up using the manual UP function.
  1. With the ignition switch ON (II), remove the No. 23 (20 A) fuse in the under-dash fuse/relay box.
  1. Turn the ignition switch OFF.
  1. After 5 seconds, install the No. 23 (20 A) fuse to the under-dash fuse/relay box.
  1. Start the engine.
  1. Move the driver's window all the way down using the manual DOWN function of the driver's power window switch.
  1. Move the driver's window all the way up using the manual UP function of the driver's power window switch, and hold it for 1 second after the window reaches the closed position.
  1. If the window does not work in AUTO, repeat steps 2 through 11.
