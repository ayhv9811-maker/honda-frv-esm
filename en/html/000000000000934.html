Timing Belt Auto-tensioner Removal and Installation
Removal
  1. Remove the timing belt.
  1. Remove the auto-tensioner.
Installation
  1. Clean the upper cover, lower cover, crankshaft, timing belt guide plate, timing belt drive pulley, and camshaft pulley.
  1. Set the crankshaft to TDC. Align the TDC mark (A) on the timing belt drive pulley with the pointer (B) on the oil pump.
  1. Set the crankshaft to top dead center (TDC). Align the TDC mark (A) on the timing belt drive pulley with the pointer (B) on the oil pump.
  1. Align the holes on the tensioner pulley (A) and tensioner base (B), then insert a 3.0 mm (0.12 in.) diameter pin into the holes.
  1. Install the tensioner spring (A) on the auto-tensioner (B).
  1. Install the auto-tensioner, then tighten the auto-tensioner mounting bolt (A) to 9.8 N·m (1.0 kgf·m, 7.2 lbf·ft).
  1. Loosen the auto-tensioner mounting bolt 180 °.
  1. Install the timing belt in a counterclockwise sequence, starting with the drive pulley.
    -1.
    Drive pulley (A)
    -2.
    Tensioner pulley (B)
    -3.
    Water pump pulley (C)
    -4.
    Camshaft pulley (D)
  1. Install the tensioner spring on the tensioner bolt.
  1. Rotate the crankshaft pulley two turns counterclockwise so that the timing belt positions on the pulleys.
  1. Set the No. 1 piston at TDC.
  1. Tighten the auto-tensioner mounting bolt (A) to 44 N·m (4.5 kgf·m, 33 lbf·ft), then remove the pin (B) from the auto-tensioner.
  1. Install the crankshaft position (CKP) sensor.
  1. Install the bolts loosely, then tighten the side engine mount bracket mounting bolts in the numbered sequence shown.
  1. Install the lower cover.
  1. Install the crankshaft pulley.
  1. Rotate the crankshaft pulley about five or six turns counterclockwise so that the timing belt positions on the pulleys.
  1. Turn the crankshaft pulley so its TDC mark lines up with the pointers.
  1. Check the camshaft pulley marks.
    • If the camshaft pulley marks are also at TDC, go to Step 20
    • If the camshaft pulley marks are not at TDC, remove the timing belt and repeat Step 2 through 18.
  1. Install the upper cover, then connect the camshaft position (CMP) sensor connector, and install the grommet.
  1. Install the upper bracket, then tighten the nuts in the numbered sequence shown.
  1. Install the cylinder head cover.
  1. Install the alternator.
  1. Install the splash shield.
