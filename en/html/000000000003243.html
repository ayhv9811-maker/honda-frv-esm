Radiator and A/C Condenser Fan, Motor, and Shroud Removal and Installation
Removal
  1. Remove the front bulkhead cover.
  1. Disconnect the fan motor connectors (A), then remove the harness clamps (B).
  1. Remove the bolt (A) securing the A/C line.
  1. Remove the A/C condenser brackets (B), then remove the bolts (C) securing the bulkhead.
  1. Remove the hood latch (A), hood latch cable clamps (B), and harness clamps (C), then remove the bulkhead (D).
  1. Remove the coolant reservoir.
  1. Remove the condenser fan shroud assembly (A), then remove the radiator fan shroud assembly (B) from condenser fan shroud side.
  1. Disassemble the fan shrouds.
Installation
  1. Assemble the fan shrouds.
  1. Install the radiator fan shroud assembly (A), then install the condenser fan shroud assembly (B).
  1. Install the coolant reservoir.
  1. Install the hood latch (A), hood latch cable clamps (B), and harness clamps (C) to the bulkhead (D).
  1. Install the bulkhead (A), then install the A/C condenser brackets (B).
  1. Install the A/C line (C).
  1. Apply body paint to the bulkhead mounting bolts.
  1. Connect the fan motor connectors (A), then install the harness clamps (B).
  1. Install the front bulkhead cover.
