NOTE: Refer to the Exploded View as needed during this procedure.
  1. Clean all the parts in solvent, dry them, and apply lubricant to all contact surfaces.
  1. Install the thrust washer (A) and needle bearing (B) onto the countershaft (C). Assemble the friction damper (D) and 1st gear (E) together and install them on the countershaft.
  1. Install the 1st gear double cone synchro (A) and synchro spring (B) on the shaft.
  1. Align the fingers on the friction damper (A) with the grooves on the 1st/2nd synchro hub (B), then install the 1st/2nd synchro hub onto the shaft (C).
  1. Install the friction damper (A) on the distance collar (B) using the special tools and a press.
  1. Install friction damper and distance collar, and the 2nd gear bearing.
  1. Install the needle bearing (A) and the 2nd gear (B), then align the holes (C) on the 2nd gear with the finger (D) on the double cone synchro.
  1. Support the countershaft (A) on a steel block (B), then install 3rd gear (C) using the special tools and a press (D).
  1. Install the 4th gear (A) using the special tools and a press (B).
  1. Install the 5th gear (A) using the special tools and a press (B).
  1. Install the needle bearing (A) using the special tools and a press (B), and install the bearing outer race.
  1. Install the ball bearing (A) using the special tools and a press (B).
  1. Securely clamp the countershaft assembly in a bench vise with wood blocks.
  1. Install the spring washer (A), and loosely install the new locknut (B).
  1. Torque the new locknut to 127 N·m (13 kgf·m, 94 lbf·ft), then loosen it and torque it again to the same value. Stake the locknut tab into the groove.
