A/T Clutch Clearance Inspection
Special Tools Required

Clutch compressor attachment
 07ZAE-PRP0100
  1. Inspect the clutch piston, discs, plates, and end plate for wear and damage, and inspect the clutch wave-plate phase difference, if necessary.
  1. Install the waved spring (A) in the 1st clutch drum (B). Starting with the clutch wave-plate, alternately install the clutch wave-plates (C) (5) and discs (D) (5), then install the clutch-end-plate (E) with the flat side down on the top disc.
  1. Install the waved spring (A) in the 2nd clutch drum (B). Install the 4.0 mm-thick plate (C) with the shear droop side (D) toward the inside of the drum, clutch disc (E), and 2.0 mm-thick flat-plate (F). Starting with the clutch disc, alternately install the discs (G) (4) and wave-plates (H) (3), then install the clutch-end-plate (I) with the flat side down on the top disc.
  1. Install the waved spring (A) in the 3rd clutch drum (B). Install the clutch flat-plate (C), clutch disc (D), and clutch flat-plate (E). Starting with the clutch disc, alternately install the discs (F) (4) and wave-plates (G) (3), then install the clutch-end-plate (H) with the flat side down on the top disc.
  1. Install the clutch pistons in the 4th and 5th clutch drums. Do not install the O-rings during inspection.
  1. Install the waved spring (A) in the 4th clutch drum (B). Starting with the clutch wave-plate, alternately install the wave-plates (C) (4) and discs (D) (4). Install the clutch-end-plate (E) with the flat side down on the top disc.
  1. Install the waved spring (A) in the 5th clutch drum (B). Starting with the clutch wave-plate, alternately install the wave-plates (C) (4) and discs (D) (4). Install the clutch-end-plate (E) with the flat side down on the top disc.
  1. Install the snap ring with a screwdriver to secure the clutch-end-plate.
  1. Set a dial indicator (A) on the clutch-end-plate (B).
  1. Zero the dial indicator with the clutch-end-plate lifted up to the snap ring (C).
  1. Release the clutch-end-plate to lower the clutch-end-plate, then put the special tool on the end-plate (A).
  1. Press the special tool down with 150−160 N (15−16 kgf, 33−35 lbf) (B) using a force gauge, and read the dial indicator (C). The dial indicator reads the clearance (D) between the clutch-end-plate and top disc (E). Take measurements in at least three places, and use the average as the actual clearance.
    Clutch-End-Plate-to-Top Disc Clearance
    Service Limit:
    1st clutch:
    1.28−1.48 mm (0.050−0.058 in.)
    2nd clutch:
    0.88−1.08 mm (0.035−0.043 in.)
    3rd clutch:
    0.93−1.13 mm (0.037−0.044 in.)
    4th clutch:
    0.93−1.13 mm (0.037−0.044 in.)
    5th clutch:
    0.93−1.13 mm (0.037−0.044 in.)

  1. If clutch clearance is out of the service limit, select a new clutch-end-plate (A) from the following tables.
