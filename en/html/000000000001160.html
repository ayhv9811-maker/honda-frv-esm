Engine Installation
  1. Install the accessory brackets and tighten their bolts to the specified torques.
  1. Attach the chain hoist to the engine/transmission as shown, then install the engine/transmission into position in the vehicle.
    Reinstall the mounting bolts/support nuts in the sequence given. Failure to follow this sequence may cause excessive noise and vibration, and reduce bushing life.
  1. Install the upper bracket (A), then tighten the nuts in the numbered sequence shown.
  1. Install the ground cable (B).
  1. Install the transmission mount (A) and the transmission mount bracket (B), then tighten the transmission mount mounting bolts.
  1. Loosely tighten the transmission mount bracket bolt (C), new support bolt (D), and support nuts (E).
  1. Remove the chain hoist from the engine/ transmission.
  1. Push the engine to the transmission side. Make sure the clearance between the side engine mount (A) and upper bracket (B) is 4.0 mm (0.16 in.) and more, then tighten the transmission mount bracket bolt/nuts in the numbered sequence shown.
  1. Raise the vehicle on the hoist to full height.
  1. Loosely tighten the new front mount mounting bolt.
  1. Tighten the new rear mount mounting bolts.
  1. Tighten the front mount mounting bolt.
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and intermediate shaft.
  1. Connect the suspension lower arm ball joints.
  1. Install the A/C compressor.
  1. Install three way catalytic converter (TWC) assembly (A). Use new gaskets (B) and new self locking nuts (C).
  1. Connect the air fuel ratio (A/F) sensor connector (A) and secondary heated oxygen sensor (secondary HO2S) connector (B), then install the harness to the clamps (C).
  1. Lower the vehicle on the hoist.
  1. Install the radiator.
  1. Raise the vehicle on the hoist to full height.
  1. Install the splash shield.
  1. Lower the vehicle on the hoist.
  1. Install the heater hoses.
  1. Install the power steering (P/S) pump, then install the P/S hose in its clamp.
  1. Install the drive belt.
  1. Install the A/C hose to the clamps.
  1. Install the evaporative emission (EVAP) canister hose (A) and brake booster vacuum hose (B).
  1. Connect the fuel feed hose, then install the quick-connect fitting cover (A).
  1. Install the select cable and shift cable with the plastic washers, washers, and new cotter pins.
  1. Install the clutch slave cylinder and clutch line bracket mounting bolt.
  1. Push the engine control module (ECM) connectors through the bulkhead, then install the grommet (A), and harness clamps (B).

  1. Install the ground cable (A).
  1. Install the battery base (B), then install the harness clamps (C).
  1. Install the battery cables (A) to the under-hood fuse/relay box, then install the harness clamp (B).
  1. Install the air cleaner housing assembly (A), then install the breather hose (B) and harness clamp (C).
  1. Connect the intake air temperature (IAT) sensor connector (D).
  1. Install the intake manifold cover.
  1. Install the front tyres/wheels.
  1. Install the battery. Clean the battery posts and cable terminals with sandpaper, then assemble them and apply grease to prevent corrosion.
  1. Check that the transmission shift into gear smoothly.
  1. Inspect for fuel leaks. Turn the ignition switch ON (II) (do not operate the starter) so the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation two or three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the engine with engine oil.
  1. Refill the transmission with transmission fluid.
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Do the ECM reset procedure.
  1. Do the crankshaft position (CKP) pattern clear/ CKP pattern learn procedure.
  1. Do the power window control unit reset procedure.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.
  1. Check the wheel alignment.
  1. Set the clock.
