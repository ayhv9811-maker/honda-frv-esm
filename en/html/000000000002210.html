Engine Installation
  1. Install the engine and accessory brackets, and tighten their bolts to the specified torques.
  1. Remove the subframe plates, and install the new subframe plates.
  1. Position the engine under the vehicle. Attach the chain hoist using the sling belt and karabiner to the engine, then lift the engine into position in the vehicle.
    • Reinstall the mounting bolts/support nuts in the sequence given. Failure to follow this sequence may cause excessive noise and vibration, and reduce bushing life.
    • Take care not to damage the A/C pipes.
  1. Install the subframe, then loosely tighten the new subframe mounting bolts.
  1. Align the reference marks (A) with the center of the subframe mounting bolts (B), then tighten the bolts to the specified torque.
  1. Lower the vehicle on the hoist.
  1. Install the side engine mount bracket (A), then loosely tighten the new bolt/nuts (B), and loosely tighten the bolt (C).
  1. Install the transmission upper mount (A), then loosely tighten the new nuts (B), and loosely tighten the new bolt (C).
  1. Install the ground cable (D).
  1. Install the transmission upper mount bracket (A), and tighten the new bolts, then loosely tighten the mounting bolt (B).
  1. Raise the vehicle on the hoist.
  1. Loosely tighten the new bolts securing the transmission lower mount.
  1. Loosely tighten the lower torque rod mounting bolt.
  1. Lower the vehicle on the hoist.
  1. Tighten the side engine mount mounting bolts/nuts in the numbered sequence shown.
  1. Tighten the transmission upper mount/bracket mounting bolts/nuts in the numbered sequence shown.
  1. Install the upper torque rod (A), then tighten the new bolts in the numbered sequence shown.
  1. Install the ground cable (B).
  1. Remove the chain hoist from the engine.
  1. Raise the vehicle on the hoist to full height.
  1. Tighten the lower torque rod mounting bolt.
  1. Tighten the new bolts securing the transmission lower mount.
  1. Lower the vehicle on the hoist.
  1. Install the A/C compressor.
  1. Install the heater hoses.
  1. Install the power steering (P/S) pump (A), and the P/S hose clamp (B).
  1. Install the drive belt.
  1. Install the expansion tank (A), then install the harness clamp (B) and water bypass hoses (C).
  1. Install the intercooler.
  1. Install the lower radiator hose.
  1. Install the upper radiator hose.
  1. Install the radiator.
  1. Raise the vehicle on the hoist to full height.
  1. Install the intercooler intake pipe (A) to the turbocharger outlet hose (B), then install the intercooler intake pipe bracket (C).
  1. Loosely tighten the intercooler intake pipe mounting bolts (D). Make sure the mark (E) of the intercooler intake pipe contacts with the turbocharger outlet hose end.
  1. Tighten the intercooler intake pipe mounting bolts, and secure the turbocharger outlet hose with the band (F), then install the intercooler intake hose (G), and secure the hose with band (H).
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and intermediate shaft..
  1. Connect the suspension lower arm ball joints.
  1. Connect the stabilizer links.
  1. Install exhaust pipe A. Use new gaskets (B) and new self-locking nuts (C).
    NOTE: Tighten the self-locking nuts (C) sequentially.
  1. Install the splash shield (A) and engine under cover (B).
  1. Install the front wheels.
  1. Lower the vehicle on the hoist.
  1. Push the engine control module (ECM) connectors through the lower dashboard, then install the grommet (A), and harness clamps (B).
