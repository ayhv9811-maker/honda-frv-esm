M/T Assembly Installation
  1. Check the two dowel pins are installed in the clutch housing.
  1. Apply Super High Temp Urea Grease (P/N 08798-9002) to the release fork (A) and the release bearing (B). Install the release fork and the release bearing.
  1. Place the transmission on the transmission jack, and raise it to the engine level.
  1. Install the three lower transmission mounting bolts.
  1. Install the clutch cover (A) and engine stiffeners (B).
  1. Install the transmission rear mount bracket (A) and the transmission rear mount (B), then loosely tighten the rear mount bracket mounting bolt (C).
  1. Install the front engine mount.
  1. Support the subframe with a 4 x 4 x 40 in. piece of wood and a jack.
  1. Install the front suspension subframe (A) in its original position by aligning the marks (B) you made in the removal procedure.
  1. Install the three mounting bolts for the transmission rear mount.
  1. Install the transmission mount bracket (A) and the transmission mounting bolt (B).
  1. Push the engine to the transmission side. Make sure the clearance between the side engine mount (A) and upper bracket (B) is 4.0 mm (0.16 in.) and more, then tighten the transmission mount bracket bolt/nuts in the numbered sequence shown.
  1. Tighten the rear mount bracket mounting bolt.
  1. Install the front engine mount bracket mounting bolt.
  1. Install the driveshafts.
  1. Install exhaust pipe A.
  1. Install the splash shield.
  1. Install the two upper transmission mounting bolts.
  1. Remove the chain hoist.
  1. Connect the vehicle speed sensor (VSS) connector.
  1. Install the cable bracket (A) and cables (B). Turn the shift lever boots (C) so the hole is facing down.
  1. Install the starter motor.
  1. Connect the starter cables (A) to the starter. Make sure that the crimped side of the ring terminal is facing out.
  1. Connect the back-up light switch connector.
  1. Apply Urea Grease UM264 (P/N 41211-PY5-305) to the end of the slave cylinder rod. Install the slave cylinder. Take care not to bend the clutch line.
  1. Connect the transmission ground cable (A), then install the clutch line bracket (B).
  1. Install the intake air duct.
  1. Install the intake resonator.
  1. Install the battery. Connect the positive (+) cable first, then the negative (âˆ’) cable to the battery.
  1. Refill the transmission fluid.
  1. Test-drive the vehicle.
  1. Check the clutch operation.
  1. Check the transmission for noise and smooth operation.
  1. Check the front wheel alignment.
  1. Enter the anti-theft codes for the radio, then enter the customer's radio station presets.
