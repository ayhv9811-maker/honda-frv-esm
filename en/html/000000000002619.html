PTC Heater Core Replacement
Do not apply current when core is removed
  • To prevent electrocution because current will flow through the whole PTC heater core.
  • If any foreign material is stuck, a stench, smoke or fire could occur.

NOTE: LHD type is shown, RHD type is symmetrical.
  1. Remove the dashboard, refer to the Shop Manual, P/N 62SJD00 .
  1. Remove the harness clip (A). Disconnect the connector (B), then remove the connector holder (C).
  1. Remove the self-tapping screws and the PTC heater core (A) from the heater unit.
  1. Install the core in the reverse order of removal. Do not pull the PTC heater core harness from a connection to prevent a circuit malfunction or harness burning.
