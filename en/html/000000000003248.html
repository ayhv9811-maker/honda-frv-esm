Thermostat Housing Installation
  1. Install the new rubber seal (A) onto the thermostat, then install the thermostat (B) with pin (C) up, and install the thermostat cover (D).
  1. Install the exhaust gas recirculation (EGR) valve (E) with a new gasket (F).
  1. Install the thermostat housing (A), using a new gasket (B) and new O-ring (C).
  1. Connect the EGR connector.
  1. Install the EGR pipe (A), using the new gaskets (B).
  1. Install the lower radiator hose (A), heater hose (B), and water bypass hoses (C), then install the harness bracket (D).
  1. Install the air cleaner housing assembly.
  1. Refill the radiator with engine coolant, and bleed the air from the cooling system with the heater valve open.
