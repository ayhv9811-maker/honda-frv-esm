Cylinder Head Installation
Install the cylinder head in the reverse order of removal:
  1. Clean the cylinder head and block surface.
  1. Install the cylinder head gasket (A) and dowel pins (B) on the engine block. Always use a new cylinder head gasket.
  1. Set the crankshaft to top dead center (TDC). Align the TDC mark (A) on the timing belt drive pulley with the pointer (B) on the oil pump.
  1. Set the camshaft to TDC. The ‘‘UP'' mark (A) on the camshaft pulley should be at the top, and align the TDC marks (B) on the camshaft pulley with the top edge of the head.
  1. Install the cylinder head on the block.
  1. Apply engine oil to the threads and under the bolt heads of all the cylinder head bolts.
  1. Tighten the cylinder head bolts sequentially in three steps.
    1st Step Torque:
    20 N·m (2.0 kgf·m, 14 lbf·ft)
    2nd Step Torque:
    49 N·m (5.0 kgf·m, 36 lbf·ft)
    3rd Step Torque:
    67 N·m (6.8 kgf·m, 49 lbf·ft)

    Use a beam-type torque wrench. When using a preset-type torque wrench, be sure to tighten slowly and do not overtighten. If a bolt makes any noise while you are torquing it, loosen the bolt and retighten it from the 1st step.
  1. Install the timing belt.
  1. Install the exhaust manifold.
  1. Install the intake manifold.
  1. Install the connecting pipe (A) with new O-ring (B).
  1. Install the upper radiator hose (C), lower radiator hose (D), and heater hose (E).
  1. Install the ground cables (A), harness clamp (B), and harness clamp bracket mounting bolt (C).
  1. Install the evaporative emission (EVAP) canister hose (A).
  1. Connect the fuel feed hose, then install the quick-connect fitting cover (B).
  1. Install the air cleaner housing.
  1. Install the resonator (A), then install the harness clamp (B).
  1. Install the battery. Clean the battery posts and cable terminals with sandpaper, then assemble them and apply grease to prevent corrosion.
  1. Inspect for fuel leaks. Turn the ignition switch ON (II) (do not operate the starter) so the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation two or three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Do the power window control unit reset procedure.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.
  1. Set the clock.
