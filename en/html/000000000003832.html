Sunroof Control Unit (Motor) Input Test
If the sunroof works OK manually, but will not work in AUTO, or reverse frequently (pinch detection), do the sunroof calibration before proceeding with the input test.
  1. Turn the ignition switch OFF.
  1. Remove the headliner.
  1. Disconnect the 10P connector (A) from the sunroof control unit (B).
  1. Inspect the connector and socket terminals to be sure they are all making good contact.
    • If the terminals are bent, loose or corroded, repair them as necessary, and recheck the system.
    • If the terminals look OK, go to Step 5 .
  1. With the control unit still disconnected, make these input tests at the connector.
    • If any test indicates a problem, find and correct the cause, then recheck the system.
    • If all the input tests prove OK, the control unit must be faulty; replace the control unit.
