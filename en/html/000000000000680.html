  1. Install the differential assembly (A) and magnet (B) in the clutch housing (C).
  1. Install the 28 mm spring washer (A) and 28 mm washer (B) over the ball bearing (C). Note the installation direction of the spring washer (A).
  1. Apply vinyl tape the mainshaft splines (D) to protect the seal. Install the mainshaft and countershaft (E) into the shift forks (F), and install them as an assembly.
  1. Install the reverse idler gear (A) and reverse gear shaft (B).
  1. Install the reverse shift fork (A) in the clutch housing with the 5th/reverse shift piece pin (B) positioned in the slot of the reverse shift fork.
  1. Install the reverse lock cam.
  1. Install the oil seal (A) using the special tools.
  1. Install the select lever (B), 22 x 32 x 1 mm washer (C), and springs (D) on the select arm (E)
  1. Install the select arm and spring pin (F).
  1. Apply liquid gasket (P/N 08C70-K0234M) to the threads of the select spring bolt, then install the select spring bolt.
  1. Select the proper size 78 mm shim (A) according to the measurements made during the mainshaft thrust clearance adjustment. Install the oil gutter plate (B), oil guide plate M, and 78 mm shim into the transmission housing (C).
  1. Remove the dirt and oil from the tramsmission housing sealing surface. Apply liquid gasket (P/N 08C70-K0234M) to the sealing surface. Be sure to seal the entire circumference of the bolt holes to prevent oil leakage.
    NOTE: If 5 minutes have passed after applying liquid gasket, reapply it and assemble the housings. Allow it to cure at least 20 minutes after assembly before filling the transmission with oil.
  1. Install the 14 x 20 mm dowel pins.
  1. Set the stopper ring (A) as shown. Place the transmission housing (B) over the clutch housing, being careful to line up the shafts.
  1. Lower the transmission housing the rest of the way as you expand the 56 mm snap ring (A). Release the snap ring so it seats in the groove of the countershaft bearing.
  1. Check that the 56 mm snap ring is securely seated in the groove of the countershaft bearing.

  1. Install the select lever (A) and shift arm shaft (B) counterclockwise, then insert the shift arm shaft.

  1. Apply liquid gasket (P/N 08C70-K0234M) to the threads, and install the interlock bolt (E).
